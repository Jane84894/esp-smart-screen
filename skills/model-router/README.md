# Model Router 使用指南

## 🎯 功能说明

自动根据任务类型选择最优模型，节省 40-60% API 请求额度。

## 📊 路由规则

| 任务类型 | 关键词示例 | 选用模型 |
|----------|-----------|----------|
| **编程开发** | 代码、函数、脚本、debug、python | `qwen3-coder-plus` |
| **复杂分析** | 分析、优缺点、为什么、方案对比 | `qwen3-max-2026-01-23` |
| **长文写作** | 文章、报告、故事、一篇 | `MiniMax-M2.5` |
| **简单对话** | 你好、谢谢、再见、在吗 | `glm-4.7` |
| **网络搜索** | 搜索、查找、最新、今天 | `qwen3.5-plus` |
| **智能家居** | 打开、关闭、灯、空调 | `qwen3.5-plus` |
| **其他** | 不匹配上述规则 | `qwen3.5-plus` (默认) |

## 🧪 测试命令

```bash
cd ~/.openclaw/workspace/skills/model-router

# 测试编程任务
node scripts/router.mjs "帮我写个 Python 函数"

# 测试简单对话
node scripts/router.mjs "你好啊"

# 测试复杂分析
node scripts/router.mjs "分析一下这个方案的优缺点"

# 测试长文写作
node scripts/router.mjs "帮我写一篇文章"

# 测试智能家居
node scripts/router.mjs "打开客厅的灯"
```

## 🔧 自定义路由规则

编辑 `config/routes.json`：

```json
{
  "routes": [
    {
      "name": "my_custom_route",
      "model": "qwen3.5-plus",
      "triggers": {
        "keywords": ["你的关键词"],
        "patterns": ["正则表达式"]
      }
    }
  ]
}
```

## ⚠️ 注意事项

1. **子代理开销**：每次 spawn 子代理会有额外开销
2. **缓存未实现**：当前版本不支持缓存
3. **学习功能未实现**：需要手动优化规则
