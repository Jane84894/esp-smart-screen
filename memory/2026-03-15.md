## ESPHome 配置问题解决
- 用户在使用 ESP-IDF 框架时遇到分区表文件缺失错误：`Missing partition table file /data/build/smart-pda/huge_app.csv`
- 分析发现 ESP-IDF 框架下 `platformio_options` 中的 `board_build.partitions` 不会自动创建分区表文件
- 提供了三种解决方案：
  1. 使用 ESP-IDF 内置分区表 (`CONFIG_PARTITION_TABLE_SINGLE_APP: "y"`)
  2. 创建自定义分区表文件并引用
  3. 使用 PlatformIO 的内置分区表
- 用户坚持使用 ESP-IDF 框架而非 Arduino 框架
